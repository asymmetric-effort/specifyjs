# AnalogClock

**Category:** Data Display  
**Path:** `components/data/analog-clock`

## Overview

Renders a live SVG clock face with configurable format,
size, date display, and timezone support.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `format` | `'12h' | '24h'` | Display format: '12h' or '24h' — in 24h, show 0-23 hour numbers |
| `size` | `number` | Clock face size in pixels |
| `showDate` | `boolean` | Show date below the clock |
| `dateFormat` | `'short' | 'long' | 'iso'` | Date format: 'short' (MM/DD/YYYY), 'long' (Month DD, YYYY), 'iso' (YYYY-MM-DD) |
| `showSeconds` | `boolean` | Show second hand |
| `timezone` | `string` | Custom timezone |
| `className` | `string` | CSS className |
