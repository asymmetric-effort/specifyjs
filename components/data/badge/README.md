# Badge

**Category:** Data Display  
**Path:** `components/data/badge`

## Overview

Count or dot indicator, positioned as an overlay on top-right
of children, or rendered inline when no children are provided.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `count` | `number` | Numeric count to display |
| `max` | `number` | Maximum count before showing "N+" |
| `dot` | `boolean` | Show a dot instead of a count |
| `color` | `string` | Badge color |
| `size` | `'sm' | 'md' | 'lg'` | Size preset |
| `variant` | `'solid' | 'outline'` | Visual variant |
| `children` | `unknown` | Child element to overlay the badge on |
