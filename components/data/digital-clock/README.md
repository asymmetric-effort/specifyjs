# DigitalClock

**Category:** Data Display  
**Path:** `components/data/digital-clock`

## Overview

A real-time digital clock component with configurable
format, timezone, date display, and seconds visibility.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `format` | `'12h' | '24h'` | Display format: '12h' for AM/PM or '24h' for 24-hour |
| `showDate` | `boolean` | Show date display |
| `dateFormat` | `'short' | 'long' | 'iso'` | Date format: 'short' (MM/DD/YYYY), 'long' (Month DD, YYYY), 'iso' (YYYY-MM-DD) |
| `showSeconds` | `boolean` | Show seconds |
| `timezone` | `string` | Custom timezone (e.g., 'America/New_York') |
| `className` | `string` | CSS className |
