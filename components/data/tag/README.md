# Tag

**Category:** Data Display  
**Path:** `components/data/tag`

## Overview

Rounded pill/chip element with optional remove button, icon,
and click interaction.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `label` | `string` (required) | Tag label text |
| `color` | `string` | Color theme (CSS color or named color) |
| `variant` | `'solid' | 'outline' | 'subtle'` | Visual variant |
| `size` | `'sm' | 'md' | 'lg'` | Size preset |
| `removable` | `boolean` | Show remove/close button |
| `onRemove` | `() => void` | Remove button handler |
| `icon` | `unknown` | Leading icon element |
| `onClick` | `() => void` | Click handler (makes tag interactive) |
| `disabled` | `boolean` | Disabled state |
