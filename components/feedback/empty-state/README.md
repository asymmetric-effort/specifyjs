# EmptyState

**Category:** Feedback  
**Path:** `components/feedback/empty-state`

## Overview

Empty content placeholder with icon, title, description, and CTA.
Renders a centered layout with a large icon or image, a title,
descriptive text, and an optional call-to-action button.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `icon` | `string` | Large icon or emoji displayed at the top |
| `title` | `string` | Title text |
| `description` | `string` | Description text |
| `action` | `EmptyStateAction` | Call-to-action button |
| `image` | `string` | Image URL displayed instead of / above the icon |
