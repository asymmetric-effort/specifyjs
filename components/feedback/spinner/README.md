# Spinner

**Category:** Feedback  
**Path:** `components/feedback/spinner`

## Overview

Loading spinner indicator.
Renders a rotating SVG circle with a stroke-dasharray gap.
Injects an inline style element for the @keyframes rotation animation.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `size` | `'sm' | 'md' | 'lg' | number` | Spinner size: preset name or pixel number |
| `color` | `string` | Spinner color |
| `thickness` | `number` | Stroke thickness in pixels |
| `speed` | `'slow' | 'normal' | 'fast'` | Animation speed |
| `label` | `string` | Accessible label for screen readers |
