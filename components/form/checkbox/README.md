# Checkbox

**Category:** Form  
**Path:** `components/form/checkbox`

## Overview

Custom styled checkbox with label.
Uses a hidden native input + visible styled div with checkmark/indeterminate indicator.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `checked` | `boolean` (required) | Whether checkbox is checked |
| `onChange` | `(checked: boolean) => void` (required) | Change handler |
| `label` | `string` | Label text |
| `indeterminate` | `boolean` | Indeterminate state (overrides checkmark display) |
| `disabled` | `boolean` | Disabled state |
| `error` | `string` | Error message |
| `size` | `'sm' | 'md' | 'lg'` | Size variant |
