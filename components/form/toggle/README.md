# Toggle

**Category:** Form  
**Path:** `components/form/toggle`

## Overview

Toggle/switch component with sliding pill animation.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `checked` | `boolean` (required) | Whether toggle is on |
| `onChange` | `(checked: boolean) => void` (required) | Change handler |
| `label` | `string` | Label text |
| `labelPosition` | `'left' | 'right'` | Label position relative to toggle |
| `disabled` | `boolean` | Disabled state |
| `size` | `'sm' | 'md' | 'lg'` | Size variant |
| `onColor` | `string` | Track color when on |
| `offColor` | `string` | Track color when off |
