# Toolbar

**Category:** Navigation  
**Path:** `components/nav/toolbar`

## Overview

Horizontal toolbar with button groups.
Renders a row of buttons, separators (vertical lines), and spacers (flex-grow).
Supports size and variant props for consistent toolbar styling.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `items` | `ToolbarItem[]` (required) | Toolbar items |
| `size` | `ToolbarSize` | Size (default: 'md') |
| `variant` | `ToolbarVariant` | Visual variant (default: 'flat') |
