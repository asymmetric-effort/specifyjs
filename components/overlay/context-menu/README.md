# ContextMenu

**Category:** Overlay  
**Path:** `components/overlay/context-menu`

## Overview

Right-click context menu with nested submenus and keyboard navigation.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `items` | `ContextMenuItem[]` (required) | Menu item definitions |
| `children` | `unknown` | Trigger area children |
