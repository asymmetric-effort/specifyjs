# Tooltip

**Category:** Overlay  
**Path:** `components/overlay/tooltip`

## Overview

Lightweight hover tooltip with configurable placement and delay.

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `text` | `string` (required) | Tooltip text content |
| `placement` | `TooltipPlacement` | Placement relative to the trigger |
| `delay` | `number` | Delay in ms before showing (default 200) |
| `maxWidth` | `string` | Max width of the tooltip |
| `children` | `unknown` | Trigger element(s) |
